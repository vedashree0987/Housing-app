# 🏠 House Price Prediction

This project uses Machine Learning to predict house prices based on features like:
- Square Footage
- Number of Bedrooms & Bathrooms
- Lot Size
- Garage Size
- Neighborhood Quality
- Year Built

## 📊 Models Used
- Linear Regression
- KNN Regression

## 📈 Evaluation Metrics
- MAE
- MSE
- RMSE
- MAPE
- R² Score
